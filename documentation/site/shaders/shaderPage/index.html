    ☁️ Noise Shaders

    Procedural textures using hash-based value, gradient, and fractal noise.
    Grouped by functional type for clarity:

    Core Modules

    Unified entry point and hash support.

    • noise.glsl — Main include file, collects all noise types
    • hash.glsl — Random number and hash utilities

    Classic Noise

    Basic, smooth procedural noise generators.

    • 1D_noise.glsl — 1D interpolated value noise
    • 2D_noise.glsl — 2D grid-based value noise
    • 3D_Perlin_noise.glsl — Classic 3D gradient noise
    • simplex_noise.glsl — Fast, low-artifact simplex noise

    Utility Noise

    Variants and helpers used for animation or visual variation.

    • 3D_noise.glsl — Time-varying pseudo 3D gradient noise
    • grayScale_noise.glsl — Grayscale noise helper

    Spatial Noise

    Noise patterns structured around space and proximity.

    • cell_noise.glsl — 2D Voronoi cell structure with jitter

    Fractal Noise

    Multi-octave patterns for natural surfaces.

    • fbm.glsl — Fractal Brownian Motion

    Noise-based Effects

    Visual effects built using the above noise modules.

    • stylized_glow_and_star_shape.glsl — Stylized star shape and bloom
    • TIE Fighter_noise.glsl — Procedural engine trail + noise shading
